Androids use 10 digit decimal formated ip address these scripts will help you convert to and from IPv4 IP address and 10digit formated IP address.

dec-to-ip.py 

Converts 10digit decimal formated ipaddress to normal ipaddress.

By default it ask for user input. However, you can comment this out and uncomment the first section that opens up a file and reads in 10 digit decimal formated ip address and converts them to normal ip address. 

ip-to-dec.py

Converts Normal ip address to 10 digit decimal mormated ipaddress.
